NCSLogger  1.0

Description

NCSLogger is a web based application with database backend to store call sign information and aid in performing net control operations in both HF and VHF amature radio bands. A formal, or directed net has a single net control station (NCS) that manages its operation for a given session. The NCS operator calls the net to order at its designated start time, periodically calls for participants to join, listens for them to answer (or check in ) keeps track of the roster of stations for that particular net session, and generally orchestrates the operation of the net. NCSLogger is hosted on a PI4 , using php, mysql database and the apache2 web server. All access is logged.

NCS OPERATIONS

Setup NCS

Clearing Check In Database.

  1. Click on Delete Check In List.
  2. Click on RED button Delete Check Ins.
  3. Click Main Site to Return to Search window.

Start NCS

Conditions at times make hearing check ins nearly impossible. You might not get the entire call sign but a partial. That is all we need to search the master list. Case of letters not important.

  1. Enter all or part of the call into search and hit return or click search call.
  2. Click check box next to name of call sign best fits your caller.
  3. Note: Results = your selection . This call was auto inserte into check in database

  4. Repeat as required.

  • At any time you can see the check ins collected by clicking View Check Ins.

Releasing Check Ins

  1. Click on View Check Ins.
  2. Optional: Click on the checkbox as each call is released. This only to show a green check next to name for visual confirmation on what has been completed in the list.
  3. Optional: Report the Count on view checkin page.
  4. Optional: Report duration time of the net.
  5. Optional: Print screen to pdf on browser to document the check in list.
  6. Optional: Return frequency to normal use.

Call Not Found

  • Write the info down on paper, most important is entire callsign.
  • Try a search with partial even down to single character.
  • Time Permitting enter new call with data from qrz
  • Click on Insert Call to Master List

Features

  • During search , when the  call is found and added to check in list the database field lastheard is updated with current time and date. (CST)
  • View Master show total count of call signs.
  • Master list imports text from netlogger or manual updates through web based app.
  • All transactions logged in a text file located on server.
  • Backup created weekly.
  • Unlimited simultaneous users.
  • View Master list shows the entire database and all fields .
